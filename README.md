# CraftBeerPi

## Intro

CraftBeerPi is an open source brewing controller.

## Installation

CraftBeerPi is python based and will require at last python 3.7.x

```text
sudo python3 -m pip install cbpi
```

```text
cbpi setup
```

```text
cbpi start
```
